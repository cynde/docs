# Groq API

## How to Integrate Groq API with Jan[](#how-to-integrate-groq-api-with-jan)

This guide provides step-by-step instructions on integrating the Groq API with Jan, enabling users to leverage Groq's capabilities within Jan's conversational interface.

Before proceeding, ensure you have the following:

*   Access to the Jan application
*   Groq API credentials

## Integration Steps[](#integration-steps)

### Step 1: Configure Groq API Key[](#step-1-configure-groq-api-key)

1.  Obtain Groq API Keys from your [Groq Console (opens in a new tab)](https://console.groq.com/keys) dashboard.
2.  Copy your Groq API Key and the endpoint URL you want.
3.  Navigate to the **Jan app** > **Settings**.
4.  Select the **Groq**.
5.  Insert the **API Key** and the **endpoint URL** into their respective fields.

  

![Server Setup](https://jan.ai/_next/static/media/Groq-1.7f7e5628.gif)

You can also manually edit the JSON file in `~/jan/settings/@janhq/inference-groq-extension`.

### Step 2: Start Chatting with the Model[](#step-2-start-chatting-with-the-model)

1.  Navigate to the **Hub** section.
2.  Select the Groq model you want to use.

The Groq Inference Engine is the default extension for the Jan application. All the Groq models are automatically installed when you install the Jan application.

3.  Specify the model's parameters.
4.  Start the conversation with the Groq model.

  

![Start Model](https://jan.ai/_next/static/media/Groq-2.5ac01a36.gif)

## Troubleshooting[](#troubleshooting)

If you encounter any issues during the integration process or while using Groq with Jan, consider the following troubleshooting steps:

*   Double-check your API credentials to ensure they are correct.
*   Check for error messages or logs that may provide insight into the issue.
*   Reach out to Groq API support for assistance if needed.
